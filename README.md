# Online IT Courses Catalog - README

## Огляд проєкту
Сучасний веб-сайт освітньої платформи з IT курсами. Проєкт включає головну сторінку, каталог курсів, детальну сторінку курсу та сторінку з цінами.

## Основні функції

### Головна сторінка (`home.ejs`)
- **Hero секція** - привабливий банер з основним закликом до дії
- **Карʼєрні шляхи** - напрямки курсів, які є на сайті
- **Популярні курси** - картки курсів з рейтингами та детальною інформацією
- **Компанії** - компанії, в яких працюють студенти після проходження курсів
- **Відгуки студентів** - картки з реальними відгуками
- **CTA секція** - фінальний заклик до реєстрації
  
### Каталог курсів (`courses.ejs`)
- **Трендові мови програмування** - інтерактивні кнопки з іконками
- **Система фільтрів** - детальна фільтрація за різними параметрами:
  - Рівень складності (Beginner, Intermediate, Advanced)
  - Тривалість курсу
  - План підписки
  - Тема
- **Сортування** - сортування курсів за релевантністю, популярністю
- **Адаптивна сітка** - курси у вигляді карток з детальною інформацією
- **Пошук** - фільтрація в реальному часі
- **Скидання фільтрів** - кнопка очищення всіх фільтрів
  
### Сторінка курсу (`course-details.ejs`)
- **Детальна інформація про курс** - назва, тривалість, рівень складності
- **Бічна панель** - швидке замовлення курсу
- **Програма курсу** - модулі з годинами навчання
- **Інформація про викладача** - фото, біографія, досвід
- **Сертифікат** - інформація про сертифікацію
- **Схожі курси** - рекомендаційна карусель

### Сторінка цін (`pricing.ejs`)
- **Тарифи** - Free та Pro плани з детальним описом
- **Перемикач періоду** - місяць/рік
- **Порівняння функцій** - списки можливостей для кожного тарифу
- **FAQ** - питання щодо оплати та підтримки

## Адаптивний дизайн
- **Mobile-first** підхід
- **Оптимізація для всіх пристроїв** - від 320px до 4K екранів
- **Гнучкі сітки** - CSS Grid та Flexbox
- **Мобільна навігація** - бургер-меню для малих екранів

## Дизайн система
- **Кольорова палітра**: 
  - Основний фіолетовий: `#5C50E1`
  - Фон: `#F7F6FF`, `#EEEDFF`
  - Текст: `#000000`, `#555555`
- **Шрифти**: 
  - Inter для заголовків
  - Manrope для основного тексту
- **Іконки**: SVG 

### Візуальні ефекти
- **Glass morphism** - склоподібний дизайн фільтрів та карток
- **Градієнтні фони** - динамічні фонові форми з розмиттям
- **Інтерактивні елементи** - анімації при наведенні та кліку
- **Backdrop filter** - сучасні CSS ефекти прозорості
  
## Технічні особливості
- **Чистий SCSS** - модульна архітектура
- **CSS анімації** - плавні переходи та hover-ефекти
- **Семантична розмітка** - HTML5 теги
- **Доступність** - ARIA атрибути, keyboard navigation
- **Оптимізація зображень** - responsive pictures
